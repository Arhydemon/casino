# Техническое задание

## Проект: «Дым Дым Казино и Бляди»

---

## 1. Общая информация

**Название проекта:**  
Дым Дым Казино и Бляди

**Тип проекта:**  
Учебное desktop-приложение

**Назначение проекта:**  
Разработка локального настольного приложения для Windows в рамках экзамена по объектно-ориентированному программированию.

**Основная идея:**  
Локальная игровая платформа с несколькими играми, единым балансом, статистикой и игрой против ботов.

**Команда разработки:**  
1 разработчик

**Срок реализации:**  
до 2 месяцев

**Цель проекта:**  
Создать desktop-приложение на Python (Flet), позволяющее играть в несколько игр с сохранением прогресса.

---

## 2. Цели и задачи проекта

### 2.1. Цель

Пользователь должен иметь возможность:

- играть в несколько игр;
- играть против ботов;
- использовать единый баланс;
- сохранять статистику;
- пользоваться удобным интерфейсом.

### 2.2. Основные задачи

- спроектировать архитектуру (ООП);
- реализовать хранение данных;
- реализовать игровые модули;
- реализовать баланс и статистику;
- реализовать ботов (покер);
- создать UI;
- подготовить сборку под Windows.

## 5. Функциональные требования

### 5.1. Профиль

Без регистрации.

Сохраняется:

- баланс
- количество игр
- количество побед
- общий выигрыш
- настройки звука

---

### 5.2. Баланс

- единый для всех игр
- ставка списывается до игры
- выигрыш начисляется после
- есть кнопка пополнения

---

### 5.3. Статистика

- общее количество игр
- количество побед
- общий выигрыш

---

## 6. Игровые модули

### Игры:

- Слоты
- Очко
- Рулетка
- Покер

---

### 6.1. Слоты

- выбор ставки
- анимация барабанов
- случайный результат
- расчет выигрыша
- обновление баланса

---

### 6.2. Очко

- игра против системы
- раздача карт
- подсчет очков
- выбор ставки
- определение победителя

---

### 6.3. Рулетка

- европейская (1 зеро)
- выбор ставки
- типы ставок:
  - число
  - цвет
  - чёт/нечёт
  - диапазон
- расчет выигрыша

---

### 6.4. Покер (Texas Hold’em)

**Стадии:**

- preflop
- flop
- turn
- river
- showdown

**Действия:**

- fold
- check
- call
- raise
- all-in

**Функциональность:**

- игрок + боты
- раздача карт
- определение комбинаций
- распределение банка

---
