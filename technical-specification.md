## Проект: «Дым Дым Казино»

## 1. Общая информация

**Тип проекта:**  
Учебное desktop-приложение

**Назначение:**  
Разработка локального приложения на Python с использованием Flet в рамках экзамена по ООП.

**Платформа:**  
Windows

**Режим работы:**  
Локальный (без интернета и сервера)

Создать приложение с несколькими игровыми режимами, единым состоянием, системой статистики и соблюдением принципов объектно-ориентированного программирования.

---

## 2. Архитектура приложения

### 2.1. Общие требования

В приложении должна использоваться объектно-ориентированная архитектура:

- все сущности реализованы через классы;
- используется инкапсуляция, наследование и полиморфизм;
- отсутствуют глобальные переменные;
- логика отделена от интерфейса.

---

### 2.2. Основные классы

В проекте должны быть реализованы следующие классы:

- Player
- AppState
- BaseGame (абстрактный класс)
- RouletteGame
- BlackjackGame (или другая игра)
- DatabaseManager
- ProfileRepository
- StatisticsRepository
- SettingsRepository
- AppStateService

---

### 2.3. Абстрактный класс

Все игровые режимы должны наследоваться от базового класса:

```python
from abc import ABC, abstractmethod

class BaseGame(ABC):

    @abstractmethod
    def play_round(self):
        pass

    @abstractmethod
    def calculate_result(self):
        pass

    @abstractmethod
    def reset(self):
        pass
```

      2.4. Переопределение методов

Каждая игра должна реализовать минимум 3 метода:

play_round()
calculate_result()
reset()

2.5. Сравнение объектов

В пользовательских классах должна быть реализована перегрузка операторов:

def __eq__(self, other):
    pass

def __lt__(self, other):
    pass

2.6. Геттеры и сеттеры

Для управления данными использовать свойства:

@property
def balance(self):
    return self._balance

@balance.setter
def balance(self, value):
    if value >= 0:
        self._balance = value

2.7. Статические методы

Минимум один метод должен быть реализован как @staticmethod.

Пример:

генерация случайного числа;
определение цвета в рулетке.
